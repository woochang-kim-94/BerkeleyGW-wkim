# This file contains all the references for the BerkeleyGW code in a text
# format. It gets transformed into a Fortran file "references.f90" by the
# preprocessing in "references.f90p".
#
# Instructions:
# 1) The format of this file is "REFERECE_LABEL:", where the
#    colon is mandatory.
# 2) Generate the formatted reference for a new paper by going to
#    https://citation.crosscite.org/, pasting the paper DOI, and selecting
#    "american-physics-society" as the formatting style.
# 3) After you finished editting the file, run `dev-scripts/mako_preprocess.py`
